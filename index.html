But first: what's the point?

Sexy cool computer science concepts like machine learning and data science often boil down to math.

A common mathematical operation we want to do is to minimize a function. Why?

In data science, we'll often try to guess the pattern in data; the most basic form of this is like finding a line-of-best-fit (called linear regression).

Like in anything, we'll want to know how wrong we were. We can make that notion a bit more rigorous with a cost function, which tells us how wrong we are. More on those specifics later.

So, if we want our best model, we'll want to minimize our cost function - and that's where gradient descent comes in.

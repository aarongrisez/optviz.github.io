But first: what's the point?

Sexy cool computer science concepts like machine learning and data science often boil down to math.

A common mathematical operation we want to do is to minimize a function. Why?

In data science, we'll often try to guess the pattern in data; the most basic form of this is like finding a line-of-best-fit (called linear regression).

Like in anything, we'll want to know how wrong we were. We can make that notion a bit more rigorous with a cost function, which tells us how wrong we are. More on those specifics later.

So, if we want our best model, we'll want to minimize our cost function - and that's where gradient descent comes in.

Abstract: what is gradient descent?

At any point (e.g. in 2D or 3D space, or 10D), gradient descent looks around the current point for the direction of the steepest decline. It inches towards that direction, and keeps on repeating - until it can't go down anymore!

Derivatives

The derivative of a single-variable function tells us how much its output changes when we slightly nudge the input (by a small positive value).

The bigger the derivative, the more the output changes with a nudge in the input.

The Sign of the Derivative

The sign of the derivative tells us the direction that the output changes: if it's positive, then the output increases as the input increases, while if it's negative, then the output decreases as the input decreases.

Putting It Together in 2D
